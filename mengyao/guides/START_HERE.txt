STEP 1: Start XAMPP
-------------------
1. Open XAMPP Control Panel
2. Click "Start" next to Apache
3. Click "Start" next to MySQL
4. Both should show green "Running" status

STEP 2: Copy Project
--------------------
Extract and copy the entire folder to your XAMPP htdocs directory:
- Windows: C:\xampp\htdocs\


STEP 3: Create Database
-----------------------
1. Open browser: http://localhost/phpmyadmin
2. Click "SQL" tab
3. Open database.sql file from project folder
4. Copy ALL the SQL code
5. Paste into phpMyAdmin SQL box
6. Click "Go" button
7. Verify "amc_db" appears in left sidebar

STEP 4: Configure Base URL
---------------------------
IMPORTANT: Update the BASE_URL to match your folder name!

1. Open: config/config.php
2. Find: $base = '/projectDraft';


STEP 5: Open the Website
-------------------------
1. Open browser
2. Go to: http://localhost/projectDraft/pages/login.php
3. You should see a login page, login and navigate between the different pages

STEP 2: UNDERSTAND THE CODE
---------------------------
Start with these files in order:
1. pages/login.php - login form 
2. pages/requestAccount.php - registration form where users requests for an account
3. pages/adminDashboard.php - user requests and existing users will be shown
                            - admin can assign the user request a role before approving/rejecting it
                            - when it is approved, the user request gets deleted from the user_requests table; when it is rejected, the user request gets deleted from the user_requests table
                            - admin can edit an existing user account by clicking on the edit button and gets directed to editusers.php
4. pages/editusers.php - edit form for existing user accounts
5. pages/process_request - processes what happens when admin click on approve/reject for user requests
6. pages/logout.php - redirects admin back to login page after clicking logout
7. css/style.css - Styling

All files have inline comments explaining the code!
